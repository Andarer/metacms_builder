# Changelog

All notable changes to MetaCMS Builder will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Dashboard screen implementation
- Projects management interface
- State management setup with Riverpod
- Firebase integration
- Testing framework setup

### Changed
- Updated Flutter to 3.16.0
- Improved theme system
- Optimized build configuration

### Fixed
- Navigation routing issues
- Theme switching bugs

## [0.1.0] - 2024-06-11

### Added
- Initial project setup
- Flutter app skeleton
- Material Design 3 theme
- Multi-language support (English, Russian)
- GitHub Actions CI/CD workflows
- Docker configuration
- Project structure and documentation
- Dependencies:
  - GetX for state management and navigation
  - Riverpod for reactive programming
  - Dio for HTTP client
  - Hive for local storage
  - Flutter ScreenUtil for responsive design
  - Google Generative AI for AI features
  - GitHub package for GitHub integration
  - Firebase for analytics

### Infrastructure
- GitHub Pages deployment configuration
- Docker containerization with Nginx
- Flutter build and deploy workflows
- README and documentation
